Manual Sidebar Verification - Fab O.S Platform

Test Objective

Verify that the sidebar in the Fab O.S platform is displaying correctly with proper formatting, styling, and all navigation elements visible.

Step 1: Navigate to the Application

URL: http://localhost:8080

Open this URL in a new browser tab/window

Step 2: Login to the Application

Use the following credentials:

  • Email: admin@steelestimation.com
  • Password: Admin@123

Step 3: Take Screenshots

Once logged in, capture screenshots of the following:

  1. Full page view - Shows the entire layout with sidebar
  2. Sidebar close-up - Focus on just the sidebar area
  3. Logo area - Top of the sidebar showing the Fab O.S logo
  4. Navigation menu - All menu items in the sidebar

Windows: Use Win + Shift + S or Snipping Tool
Mac: Use Cmd + Shift + 4
Linux: Use Screenshot tool or PrtScn

Verification Checklist

Please verify the following aspects of the sidebar:

  • Width: Should be exactly 250px wide
  • Background Color: Should be dark blue (#0d1a80)
  • Logo: Fab O.S logo should be visible at the top
  • Navigation Items: All menu items should be visible and properly styled
  • Text Color: White or light colored text for good contrast
  • Icons: FontAwesome icons should be visible next to menu items
  • Hover Effects: Menu items should have hover effects
  • Active State: Current page should be highlighted in the menu
  • Spacing: Proper padding and margins between elements
  • Responsiveness: Sidebar should stay fixed when scrolling

Expected Navigation Menu Items

  • Dashboard (Home icon)
  • Estimations
  • Customers
  • Projects
  • Templates
  • Reports
  • Settings (if admin)
  • User Profile (bottom)
  • Logout (bottom)

Screenshot Location

Save your screenshots to: /mnt/c/Fab.OS Platform/Fab O.S/

Suggested naming:

  • sidebar-full-page-[timestamp].png
  • sidebar-closeup-[timestamp].png
  • sidebar-logo-[timestamp].png
  • sidebar-nav-[timestamp].png

Known Issues to Check

  • Sidebar might be collapsed or hidden on mobile view
  • CSS might not be loading properly (check browser console)
  • Navigation items might be missing or improperly styled
  • Logo image might be broken or missing

What Success Looks Like

A properly formatted sidebar should have:

  • ✓ Clean, professional appearance with dark blue background
  • ✓ Clear, readable white text
  • ✓ Visible Fab O.S logo at the top
  • ✓ All navigation items properly aligned and spaced
  • ✓ Icons displaying correctly next to menu items
  • ✓ Smooth hover transitions on menu items
  • ✓ Fixed position that doesn't move when scrolling
